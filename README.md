# pact-thinking

An OpenClaw skill for structured thinking with the P.A.C.T. framework:

- **P — Premise**: surface the current view
- **A — Attack**: stress-test it before proposing alternatives
- **C — Compare**: compare multiple paths with explicit trade-offs
- **T — Take**: organize the decision, while leaving the final choice to the user

## Files

- `SKILL.md` — skill definition and operating instructions
- `references/prompts.md` — reusable prompt patterns

## Intended use

Use this skill when the user wants:

- clearer judgment
- structured debate
- assumption testing
- option comparison
- a reusable thinking framework

## Installation

Copy this skill directory into your OpenClaw skills path and activate it when a task matches the description in `SKILL.md`.
